# 🛒 متتبع الطلبيات اليومية

تطبيق ويب لتتبع الطلبيات اليومية وإعلانات المنتجات وعرض الإحصائيات.

## المميزات

- **📦 الطلبيات** — إدخال طلبية يومية باختيار التاريخ والمنتج مع مجموع اختياري
- **📢 الإعلانات** — تسجيل المنتجات التي تشتغل إعلاناتها يومياً مع مبلغ الإنفاق
- **📊 الإحصائيات** — ترتيب المنتجات من الأحسن للأسوأ مع فلاتر تاريخ، مع تنبيهات ملونة:
  - 🔴 **أحمر**: منتج مرّ عليه 3 أيام متتالية إعلاناته تشتغل بدون أي طلبية
  - 🟠 **برتقالي**: منتج حصل على طلبيتين أو أقل في الفترة المحددة
- **⚙️ الإعدادات** — إضافة وحذف المنتجات

## التثبيت والتشغيل

```bash
# 1. تثبيت الاعتمادات
npm install

# 2. تشغيل التطبيق محلياً
npm start

# 3. بناء نسخة الإنتاج
npm run build
```

## النشر على GitHub Pages

```bash
# تثبيت أداة النشر
npm install --save-dev gh-pages

# إضافة هذا السطر في package.json تحت "scripts":
# "predeploy": "npm run build",
# "deploy": "gh-pages -d build"

# وإضافة هذا في أعلى package.json:
# "homepage": "https://<username>.github.io/<repo-name>"

# ثم نشر
npm run deploy
```
